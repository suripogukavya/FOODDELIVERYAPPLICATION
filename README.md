# 🍔 Food Delivery Application

## 📌 Overview
The Food Delivery Application is a dynamic web-based project developed using Java technologies.  
It allows users to log in, explore available restaurants, and interact with the system efficiently.

This project demonstrates core backend concepts like Servlets, JSP, and session management along with a simple and responsive user interface.

---

## 🚀 Features
- 🔐 User Login Authentication
- 🍽️ Restaurant Listing Page
- 🔄 Session Management
- 🎨 Responsive and Attractive UI
- ⚡ Fast and Dynamic Navigation

---

## 🛠️ Tech Stack
- **Backend:** Java (Servlets, JSP)
- **Database:** MySQL
- **Frontend:** HTML, CSS
- **Server:** Apache Tomcat

---

## ▶️ How to Run the Project
1. Clone the repository
2. Import into Eclipse IDE
3. Configure Apache Tomcat Server
4. Run the project on server
5. Open in browser:(http://localhost:8080/FOODDELIVERYAPPLICATION/)


---

## 📂 Project Structure
- `Servlets` → Handles business logic
- `JSP` → UI Pages
- `DAO` → Database operations
- `Model` → Data objects
